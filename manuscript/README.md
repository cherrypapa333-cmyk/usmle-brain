# Manuscript Working Folder

Files in this folder are draft scaffolding for the review manuscript:

- **`music-therapy-hf-review.md`** — full manuscript draft (Introduction → Conclusion), including suggested co-author section assignments, conceptual figure description, and AMA reference list.
- **`summary-table.md`** — updated evidence table with an added PNIE-mechanism column.

## Important policy reminder for co-authors

Per submission rules, the final manuscript must contain **less than 25% AI-generated content** as measured by AI-detection tools (e.g., zerogpt.com). The drafts in this folder are intentionally section-modular so that each author can rewrite their assigned section in their own voice. Please do not paste the draft as-is into a submission.

## Suggested workflow

1. Each author opens `music-therapy-hf-review.md` and locates their assigned section(s).
2. Rewrite that section in original prose, retaining citation numbers but reshaping phrasing and argument structure.
3. Verify every reference and DOI against PubMed.
4. Run the merged draft through an AI-detection tool; revise any section above threshold.
5. Convert the ASCII conceptual figure to a vector graphic before submission.
